# Codex Instructions

This repository is intentionally minimal. Prefer small, focused changes and keep
new project conventions documented here as they are introduced.

## Development

- Check the working tree before editing: `git status --short --branch`.
- Use `rg` for searching files and text.
- Keep generated artifacts, build output, dependency directories, and local
  secrets out of Git.
- Add build, test, and formatting commands to this file once the project stack is
  chosen.

## Verification

There is no project-specific build or test command yet. When one is added,
document the command here and run it before finalizing code changes.
